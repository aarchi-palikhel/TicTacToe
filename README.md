# Tic-Tac-Toe

A classic Tic-Tac-Toe game built with React.

## About

This project was built by following the official React tutorial - a great hands-on introduction to React fundamentals including components, state, and props.

## Credits

Tutorial: [Tutorial: Tic-Tac-Toe – React](https://react.dev/learn/tutorial-tic-tac-toe)  
Source: [react.dev](https://react.dev)

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

```bash
npm install
```

### Running the app

```bash
npm start
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
